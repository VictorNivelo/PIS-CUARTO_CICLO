Guía de Uso del Sistema

Primeros Pasos

Bienvenido al sistema de predicción de deserción estudiantil de la Universidad Nacional de Loja. Para comenzar a utilizar el programa, sigue estos pasos:

  1. Accede al Formulario: Dirígete a la página donde se encuentra el formulario para ingresar los datos de los estudiantes.
  2. Completa los Campos del Formulario: Llena cada campo con la información requerida sobre los estudiantes. Asegúrate de ingresar datos válidos y seguir las instrucciones proporcionadas para cada campo.
  3. Revisa los Mensajes de Error: Si alguno de los campos tiene un error (como un valor no válido o un dato fuera de rango), se mostrará un mensaje de advertencia debajo del campo correspondiente.
  4. Envía el Formulario: Cuando todos los datos estén correctos, haz clic en el botón "Crear datos" para enviar el formulario.
  5. Verifica el Mensaje de Éxito: Después de enviar el formulario, se mostrará un mensaje de éxito si los datos se han guardado correctamente.

Modelo Matemático y Representación Gráfica

Este sistema se basa en un modelo matemático avanzado para analizar y predecir la deserción estudiantil. Las principales características incluyen:

  • Modelo Matemático: El sistema utiliza algoritmos y fórmulas matemáticas para procesar y analizar los datos ingresados. Esto asegura que las predicciones sean precisas y las conclusiones sean confiables.
  • Representación Gráfica: Los resultados se representan visualmente mediante gráficos y diagramas, facilitando la interpretación de las predicciones. Los gráficos pueden ser de distintos tipos, como barras, líneas o sectores, según el tipo de análisis realizado.
  • Descarga de Reportes: Una vez generado el reporte basado en el análisis, puedes descargarlo en formato PDF o Excel para su revisión o impresión. Esto te permite tener un registro físico o digital de las predicciones realizadas.

Botones y Funciones del Formulario

A continuación, se detallan los botones disponibles en el formulario y sus respectivas funciones:

Este botón permite regresar a la página anterior. Ideal para cuando necesitas retroceder después de una acción sin guardar cambios. Si no hay una página anterior registrada, redirige a la página principal.

Utiliza este botón para crear nuevos registros o datos en el sistema. Asegúrate de llenar todos los campos necesarios antes de hacer clic en "Crear".

El botón "Buscar" te permite buscar información específica dentro del sistema. Introduce los criterios de búsqueda y haz clic en el botón para ver los resultados.

Este botón es utilizado para eliminar registros del sistema. Asegúrate de confirmar la eliminación, ya que esta acción no puede deshacerse.

El botón "Guardar" guarda los cambios realizados en el formulario o registro actual. Utiliza este botón para asegurar que tus datos se almacenen correctamente.

Usa el botón "Importar" para cargar datos desde archivos externos al sistema. Asegúrate de que el archivo esté en el formato correcto para evitar errores durante la importación.
